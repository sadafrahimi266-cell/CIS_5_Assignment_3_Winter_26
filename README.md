# CIS 5 – Week 3 Assignment

## Description
This program analyzes a fixed set of 30 student test scores using arrays and loops in C++.  
No user input is required.

## What the program does
- Stores 30 pre-filled student scores in an array
- Calculates the average score
- Finds the highest and lowest scores
- Counts how many students passed (score ≥ 60)
- Counts how many students earned an A (score ≥ 90)
- Prints all scores in original order
- Prints all scores in reverse order

## Concepts used
- Arrays
- for loops
- Conditionals (if statements)
- Basic statistics (sum, average, min, max)
- Formatted output using `<iomanip>`

## How to run
Compile and run `main.cpp` using a C++ compiler or an online IDE.
